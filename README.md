# Master in Bioinformatics Project (under development)
This repository contains the requirements, code, and backlog of my bioinformatics master's thesis developed at CIMA University of Navarra. The expected end date of the project is February 1, 2026.  

Provisional title of the project is **"Prediction of Secondary Structures of IncRNAs Associated with Colorectal Cancer Using an RNA Language Model"**
