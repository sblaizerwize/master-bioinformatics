# RNA Structure Prediction — rna-structure-prediction

Overview
--------
This folder contains scripts for RNA secondary-structure prediction and downstream analysis using RNAStructure and bpRNA tools. It processes FASTA inputs and generates 2D structure files providing CSV summaries and motifs' statistics.

Contents
--------
- `isoforms_eocrc.fasta`, `isoforms_locrc.fasta`, `6_isoforms.fasta` — FASTA input files used to produce the structure predictions.
- `fasta_to_results_rnastructure.py` — Python script used to processing isoforms in FASTA format to obtain their secondary structure using `RNAstructure`. Results are stored in CSV format: `rnastructure_eocrc.csv` and `rnastructure_locrc.csv`.
- `run_bpRNA.sh` — BASH script to obtain motifs' statistics from .db files generated by `RNAstructure` using the bpRNA toolkit. Results are stored in .st format.
- `bpRNA_st_stats.py` — Computes statistics from `.st` (secondary-structure) files and produces a CSV summay: `bpRNA_st_stats_eocrc_all.csv`, `bpRNA_st_stats_locrc_all.csv`, and `bpRNA_st_stats_6_isoforms_all.csv`.
- `2D_structure_eocrc/`, `2D_structure_locrc/`, `2D_structure_6overlapped_isoforms/` — Directories holding per-transcript files (`.ct`,`.st`,`.db`,`.svg`, `.pdf`) from each cohort generated by the proposed workflows.
- `motifs_plots.Rmd` — Rmarkdown script to generate heatmap plots of the structural content of isoforms. 
- 


Requirements
------------
- Python 3 (tested with Python 3.x)
- RNAstructure software and bpRNA toolkit. Install those tools separately and ensure they are on your `PATH` if required by the scripts.
- Make sure `run_bpRNA.sh` is executable if you plan to run it:

```bash
chmod +x run_bpRNA.sh
```

Quick usage
-----------
Below are example commands to reproduce the workflows. 

- Convert FASTA isoforms to .ct and .db files using RNAstructure:

```bash
python3 fasta_to_results_rnastructure.py isoforms_eocrc.fasta rnastructure_eocrc.csv
```

- Run bpRNA to convert all .db files in the current directory:

```bash
./run_bpRNA.sh
```

- Produce aggregated bpRNA statistics from `.st` files:

```bash
python3 bpRNA_st_stats.py input.st output.csv 
```
